About





This project is a student project by Emil Ötting at Hochschule Düsseldorf as part of the course "Interactive Storytelling" by Prof. Kay Schröder in 2025/2026. It consists of a Python-based data-gathering module, providing a solid database for the data visualization module, as well as an interactive multi-plot web application using JavaScript.



Data Collection

The underlying data are the Billboard Year-End Hot 100 singles from 1959 to 2024. Scraping Wikipedia, information about the featured songs and artist was automatically collected. This information was then cleansed and fed into the Spotify-API to return basic information on the specific song. Using the Spotify Song-ID (a unique identifier), more detailed information about each song was collected from the https://reccobeats.com/ - API. All the collected information was combined into a database as a backbone for the visualization module.

Disclaimer

Although effort was taken to avoid mix-ups during data collection and processing, some semantic errors might still occur due to the automated nature and dependency on the Spotify-API output. In some cases the requested data was also not available, leading to missing data fields.



Data Visualization

Specific aspects of the data collected were visualized over a total of 5 interactive plots. Special focus was laid on the average loudness, danceability, energy and positional development of songs scoring a chart ranking. In order to create appealing visuals, the libraries three.js and d3.js were incorporated. More detailed information about the information depicted can be found underneath the respective visualizations.
