Chart Appearance

This visual contains information about the duration of songs within the charts. Each year is represented by a vertical line consisting of ideally 100 points, each representing the song at the chart position according to its vertical index (chart-toppers at the top). If a song is chart-listed in two consecutive years, a line indicates its positional development. Additionally, the in the following year's bar, the respective dot is color red if the position decreased, green if it increased and grey in case it remained the same. In order to allow for a closer look, 2 intervals can be set, one each for year and chart position. Hovering over a data point opens up a tooltip displaying basic information on the corresponding song. Clicking on a point opens up a more detailed tooltip version.
